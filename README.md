# Unity 2D Frog Platformer

## About
This game is developed by me. <br>
As part of my learning process in Unity, I've taken an online course on Udemy. <br> 

## Game Mechanics
In this game, you play as Froppy the frog. <br>
You control him with the standard movement inputs. <br>
Froppy can move, jump, and attack (In development). <br>
You'll face several levels, each filled with enemies and collectibles. <br>
